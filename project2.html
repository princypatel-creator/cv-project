PROJECT 2

CLONE OF A WEBSITE

This project is a front-end replica of the official homepage of Flipkart, one of India's largest e-commerce platforms. Built exclusively using semantic HTML5 and custom CSS3, this project focuses entirely on replicating the user interface (UI), design aesthetics, and layouts with some frameworks like (bootstrap)or heavy scripting.
It serves as a comprehensive practice for mastering layouts, grids, alignment, and responsiveness in web design.

Key Points:--

Interactive Navigation Bar: Features the iconic blue background, search bar, corporate branding logo, and individual dropdowns for user profiles and carts.
Category Menu Slider: A visual list of top-level retail categories (Electronics, Fashion, Home, Grocery) utilizing flexbox layouts.
Banner & Promotional Carousels: Static layout structures replicating current seasonal banners and sales.
Product Grid Layouts: Neatly organized product blocks featuring product images, item descriptions, price tags, and discounts.
Hover Animations:Micro-interactions built using native CSS changes for interactive elements like buttons and text links.
