PROJECT 3

Making a Website to store data in Local Storage

A lightweight, secure, and responsive web application designed to streamline the visitor registration and gate pass issuance process for a High Court. Built entirely using frontend technologies (HTML5, CSS3, and JavaScript), the system captures visitor details, generates temporary gate passes, and securely stores logs locally within the browser’s LocalStorage for instant retrieval and administrative review without the need for a backend database.

Key Features:---

Visitor Registration Form: A structured form to collect essential security data, including visitor name, contact details, purpose of visit, case number (if applicable), and designated court room/advocate name.
Implement Local Storage: Implements JavaScript localStorage to save, update, and persist visitor logs across browser sessions, ensuring no data loss during accidental reloads.
Responsive & Secure UI: Styled with modern CSS to ensure the interface is highly readable and responsive across desktops and tablets used at security checkpoints.
