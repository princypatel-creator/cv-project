# CV Resume 📄

A clean, modern, and fully responsive digital resume website designed to showcase professional information, skills, education, and achievements in an elegant and accessible format.
The primary goal of this project is to create a professional online resume that highlights an individual's profile in a modern and accessible way. It also serves as a practical implementation of responsive web design principles and front-end development skills
This project transforms a traditional resume into an interactive web experience, allowing visitors to explore personal and professional details through a visually appealing interface. It demonstrates the use of front-end technologies to build a professional portfolio-ready resume website.

## ✨ Features

* Modern and visually appealing resume layout
* Fully responsive design for all screen sizes
* Well-structured sections for personal and professional details
* Skills, education, experience, and contact information display
* Clean and user-friendly interface

## 🛠️ Technologies Used

* **HTML5** – Semantic structure and content organization
* **CSS3** – Styling, layouts, and responsive design
* **JavaScript** – Interactive elements and enhanced user experience
